Aplicativos para Aprendizado em Libras

Abeille Libras

O Abeille Libras é um aplicativo que facilita o aprendizado de Libras, com recursos interativos e vídeos explicativos.

Funcionalidades:

  • Aulas interativas
  • Vídeos explicativos em Libras
  • Quiz para fixação do conteúdo

VLibras

VLibras é uma suíte de ferramentas para tradução de textos em Libras em sites e dispositivos móveis.

Funcionalidades:

  • Tradução automática de textos para Libras
  • Plugin para navegadores
  • Aplicativo para dispositivos móveis

Librário – Libras para Todos

O Librário é uma plataforma para o ensino e disseminação da Libras, oferecendo vídeos e material interativo.

Funcionalidades:

  • Vídeos tutoriais de Libras
  • Material didático interativo
  • Conteúdo acessível para todos os níveis

Alfabeto Libras

Este app ensina o alfabeto manual de Libras através de quizzes e vídeos tutoriais.

Funcionalidades:

  • Aprendizado do alfabeto manual de Libras
  • Quiz interativo para aprender as letras
  • Vídeos de demonstração dos sinais

Hand Tal

Hand Tal é um app para tradução e aprendizagem de sinais em Libras, promovendo a inclusão social.

Funcionalidades:

  • Tradução de sinais em Libras para texto
  • Funcionalidade de vídeo para demonstrar sinais
  • Integração com redes sociais para promover a inclusão
